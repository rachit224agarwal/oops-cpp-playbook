# CS Core OOP in C++

This repository contains my hands-on C++ Object-Oriented Programming (OOP) practice.
I built it as a long-term revision record for CS core concepts and interview preparation.

## Why This Repo
- Keep concept-wise OOP examples in one place
- Build strong CS core fundamentals through code
- Maintain a revision-friendly reference for the future

## Topics Covered
- Encapsulation
- Constructors (default, parameterized, copy)
- Destructor
- Inheritance
  - Single inheritance
  - Multilevel inheritance
  - Multiple inheritance
  - Hierarchical inheritance
- Polymorphism
- Abstraction
- Friend function
- Friend class
- Static keyword (class-level usage and behavior)

## How to Run
Compile and run any file using `g++`:

```bash
g++ constructor.cpp -o constructor
./constructor
```

Example for another topic:

```bash
g++ inheritance.cpp -o inheritance
./inheritance
```
